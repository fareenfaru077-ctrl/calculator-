# CLI Calculator

## Description

This project is a simple **Command Line Calculator** built using Python. It allows users to perform basic arithmetic operations — addition, subtraction, multiplication, and division — through an interactive menu-driven interface. The program includes input validation and error handling to ensure smooth execution and prevent crashes caused by invalid entries or division by zero.

## Features

* Perform basic arithmetic operations
* User-friendly CLI interface
* Input validation for numeric values
* Handles division by zero safely
* Continuous calculation until user exits

## Requirements

* Python 3.x

## How to Run


python calculator.py
```
